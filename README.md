# 📧 AI Cold Email Generator

Generate **job-winning cold emails instantly** using AI 🚀  
Built with **Llama 3.3 + LangChain + Streamlit**

---

## 🌟 Features

- 🔗 Extract job details from URL or text
- 🤖 AI-powered cold email generation
- 🎯 Personalized emails based on skills
- 📂 Portfolio link integration
- ⚡ Fast & clean UI using Streamlit
- ⬇️ Download generated emails

## 🧠 How It Works

1. User inputs job URL or description  
2. Web content is loaded using LangChain  
3. Text is cleaned & processed  
4. LLM extracts job role + skills  
5. Portfolio links are matched  
6. AI generates a personalized cold email  

---

## 🛠️ Tech Stack

- **Frontend:** Streamlit  
- **Backend:** Python  
- **LLM:** Llama 3.3 (via Groq)  
- **Framework:** LangChain  
- **Vector DB:** ChromaDB
